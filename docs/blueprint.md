# **App Name**: MediVault

## Core Features:

- Medical History Upload: Allow users to upload and categorize their medical documents, including prescriptions, X-rays, CT scans, MRI scans, and lab results.
- Appointment Calendar: Integrate a calendar feature to manage and display upcoming doctor appointments.
- Local Doctor Directory: Provide a searchable directory of local doctors with contact information and specializations.
- Smart Summary Generation: Leverage AI to create a smart summary that extracts relevant medical details, analyzes the records for potentially important facts, and flags these for user review. The summary is presented as a 'tool' to assist a user and provide efficiency when the user is assembling a patient history to share with providers. 

## Style Guidelines:

- Primary color: Calm blue (#72BCD4), symbolizing trust and health.
- Background color: Light blue (#E0F7FA), for a clean, airy feel.
- Accent color: Soft green (#A5D6A7) for positive reinforcement and important actions.
- Body and headline font: 'PT Sans' for a modern, readable, humanist style.
- Use clear, recognizable icons for each medical category to enhance usability.
- A clean, tab-based layout to organize different categories of medical information. Ensure the design is responsive and accessible across devices.
- Subtle transitions and loading animations to maintain user engagement without being distracting.
